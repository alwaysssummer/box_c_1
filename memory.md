# 프로젝트 작업 기록

## 📅 최근 작업: 메인 페이지 레이아웃 구조 개선 (2024-12-17)

### 🎯 작업 목표
메인 페이지의 레이아웃을 더 체계적이고 사용하기 편하게 재구성했습니다.

### ✅ 완료된 작업

#### 1. **GNB(Global Navigation Bar) 헤더 추가**
   - **파일**: `src/components/layout/Header.tsx` (새로 생성)
   - **내용**: 
     - 화면 최상단에 고정된 헤더 추가
     - 타이틀: "📚 대시보드"
     - 메인 탭 메뉴: 회원관리 / 교재관리 / 설정
   - **효과**: 전체 시스템의 주요 섹션으로 쉽게 이동 가능

#### 2. **3단 패널 레이아웃 구조 유지 및 개선**
   - **구조**:
     ```
     ┌─────────────────────────────────────┐
     │ Header (GNB)                        │
     ├──────────┬──────────────┬───────────┤
     │ Sidebar  │ MainContent  │ RightPanel│
     │ (좌측)   │ (중앙)       │ (우측)    │
     └──────────┴──────────────┴───────────┘
     ```
   - **효과**: 좌측(트리/리스트), 중앙(메인 컨텐츠), 우측(확장 기능)으로 명확하게 구분

#### 3. **LNB(Local Navigation Bar) 메뉴를 중앙 패널 상단에 가로 배치**
   - **파일**: `src/components/layout/MainContent.tsx`
   - **내용**:
     - 교재관리 탭: 현황 / 문장분리 / 문제출제 / 문제관리
     - 설정 탭: 블록 관리 / 문제 유형 / 시스템 설정
   - **효과**: 각 탭의 서브 메뉴가 중앙 패널 상단에 가로로 표시되어 접근성 향상

#### 4. **설정 > 블록 관리의 목록을 좌측으로 이동**
   - **파일**: `src/app/page.tsx`
   - **내용**: 우측 패널에 있던 블록 목록을 좌측 사이드바로 이동
   - **효과**: 좌측에서 블록을 선택하고, 중앙에서 상세 정보를 보는 직관적인 흐름

#### 5. **헤더 타이틀 변경**
   - **파일**: `src/components/layout/Header.tsx`
   - **변경**: "교재 관리 시스템" → "대시보드"
   - **효과**: 더 간결하고 명확한 타이틀

### 🔗 파일 간 관계

```
src/app/page.tsx (메인 컨테이너)
│
├─ Header.tsx (최상단 GNB)
│  └─ 회원관리 / 교재관리 / 설정 탭
│
├─ Sidebar.tsx (좌측 패널)
│  └─ 교재 트리, 블록 목록 등 동적 컨텐츠
│
├─ MainContent.tsx (중앙 패널)
│  ├─ LNB 메뉴 (상단)
│  └─ 메인 컨텐츠 (하단)
│
└─ RightPanel.tsx (우측 패널)
   └─ 확장 기능, 상세 정보 등
```

### 📌 주요 상태 관리

- `activeTab`: 현재 선택된 GNB 탭 (회원관리/교재관리/설정)
- `settingMenu`: 설정 탭의 서브 메뉴 (블록 관리/문제 유형/시스템 설정)
- `contentMode`: 교재관리 탭의 서브 모드 (현황/문장분리/문제출제/문제관리)

### 🎨 레이아웃 구조 설명

1. **최상단 (Header)**: 
   - 시스템 타이틀과 주요 섹션으로 이동하는 탭

2. **좌측 패널 (Sidebar)**:
   - 교재관리: 교재 트리 구조
   - 설정 > 블록 관리: 블록 목록

3. **중앙 패널 (MainContent)**:
   - 상단: 현재 탭의 서브 메뉴 (LNB)
   - 하단: 선택한 메뉴에 따른 메인 컨텐츠

4. **우측 패널 (RightPanel)**:
   - 확장 기능, 상세 정보 등 보조 컨텐츠

## 📚 프로젝트 개요

이 프로젝트는 교재 관리 시스템으로, 교재 등록, 문제 출제, 문제 관리 등의 기능을 제공합니다.

### 주요 기능
- 교재 관리 (그룹/교재/단원/지문 구조)
- 문장 분리 (AI 기반)
- 문제 출제 (원클릭 생성)
- 문제 관리 (필터링 및 편집)
- 설정 (블록 관리, 문제 유형, 프롬프트 등)
