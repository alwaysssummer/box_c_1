# 📦 기본 데이터 유형 가이드

> 기본 데이터 유형은 문장 분리 시 자동으로 생성되는 데이터입니다.  
> **AI 호출 없이 즉시 사용 가능**하며, 토큰 소비가 없습니다.

---

## 📋 기본 데이터 유형 목록

| ID | 이름 | 대상 | 설명 |
|----|------|------|------|
| `00000000-0000-0000-0000-000000000001` | 영어지문 | 지문 | 지문 전체 영어 원문 |
| `00000000-0000-0000-0000-000000000002` | 한글해석 | 지문 | 문장별 한글 번역을 합친 전체 해석 |
| `00000000-0000-0000-0000-000000000003` | 영어한줄 | 문장 | 문장별 영어 원문 (번호 포함) |
| `00000000-0000-0000-0000-000000000004` | 한글한줄 | 문장 | 문장별 한글 번역 (번호 포함) |

---

## 🚀 사용 방법

### 1. 기본 데이터 유형 시드 (최초 1회)

서버가 실행 중인 상태에서 API를 호출하여 기본 데이터 유형을 등록합니다.

```bash
# GET - 시드 상태 확인
curl http://localhost:3000/api/data-types/seed-base

# POST - 기본 데이터 유형 등록
curl -X POST http://localhost:3000/api/data-types/seed-base
```

또는 브라우저에서:
```
http://localhost:3000/api/data-types/seed-base
```

### 2. 데이터 생성 메뉴에서 선택

1. **교재관리** → **데이터 생성** 탭 클릭
2. **데이터 유형 선택** 드롭다운 클릭
3. **📦 기본 데이터 (AI 호출 없음)** 섹션에서 원하는 유형 선택
4. 지문 선택 후 **데이터 생성** 버튼 클릭

### 3. 설정에서 확인

1. **설정** → **데이터 유형** 탭 클릭
2. 목록에서 파란색 **기본 데이터** 배지가 붙은 유형 확인

## 📊 기본 데이터 유형별 출력 형식

### 1. 영어지문 (PASSAGE_ENGLISH)

지문 전체 영어 원문을 반환합니다.

**입력:** 지문 ID

**출력 예시:**
```
Thank you for applying for the winter internship program. I'm writing this email to provide you with important information regarding the next step in your application process. Based on the materials you submitted, we've recognized your genuine interest in advancing your technical expertise and software development skills.
```

---

### 2. 한글해석 (PASSAGE_KOREAN)

문장별 한글 번역을 합쳐서 전체 해석을 반환합니다.

**입력:** 지문 ID

**출력 예시:**
```
겨울 인턴십 프로그램에 지원해 주셔서 감사합니다. 이 이메일은 지원 과정의 다음 단계에 대한 중요한 정보를 제공하기 위해 작성되었습니다. 제출하신 자료를 바탕으로, 우리는 귀하의 기술 전문성과 소프트웨어 개발 능력 향상에 대한 진정한 관심을 인정했습니다.
```

---

## 💡 문제 유형에서 활용하기

기본 데이터 유형을 활용하여 다양한 학습자료를 만들 수 있습니다:

| 문제 유형 | 사용 데이터 유형 | 레이아웃 |
|----------|-----------------|----------|
| 한줄영어 | 영어한줄 | 번호 + 영어 문장 |
| 한줄해석 | 영어한줄 + 한글한줄 | 번호 + 영어 + 한글 |
| 좌우한줄 | 영어한줄 + 한글한줄 | 좌: 영어 / 우: 한글 |
| 완성영문 | 영어지문 | 전체 영어 지문 |

## ❓ FAQ

### Q: 기본 데이터 유형은 왜 AI 호출이 없나요?
A: 기본 데이터는 문장 분리 시 이미 생성된 데이터(지문 내용, 문장별 번역 등)를 그대로 사용하기 때문입니다.

### Q: 기본 데이터 유형을 삭제할 수 있나요?
A: 삭제는 가능하지만 권장하지 않습니다. 다시 등록하려면 `/api/data-types/seed-base` API를 호출하세요.

### Q: 기본 데이터 유형을 수정할 수 있나요?
A: 설정 > 데이터 유형에서 선택 후 수정할 수 있지만, 시스템 제공 유형이므로 수정하지 않는 것이 좋습니다.
